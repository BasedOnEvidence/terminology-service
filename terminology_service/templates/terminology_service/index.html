Instructions

The Terminology service provides a mechanism for managing the process of maintaining reference information and provides functions for its integration between various accounting and information systems, being a single source of consistent information about reference directories.

How to get data

You can use api to get data without authentication. See examples below.

  • Get list of directories: info/api/directories/
  • Get list of directories on specific page: info/api/directories/?page=2
  • Get events relevant to the specified date: info/api/directories/?date=2021-12-02
  • Get list of items in directories: info/api/items
  • Get list of items of specified directory:info/api/items/?name=Antibiotics&version=1.2.643.2.69.1.1.1.74

How to add data

Open admin panel. There are following accounts available:

  • Super user with full access. Login: admin; password: Qwerty!@
  • Administrator with full access to directories tables. Login: termadmin; password: Qwerty!@
You can add new data via provided forms.
